## 1. ¿Por qué usar Selección Múltiple?
A veces, nuestro programa debe elegir entre muchas alternativas posibles. Aunque podríamos usar varios "Si-Sino" anidados, esto puede volver el código difícil de leer y mantener. Para estos escenarios, donde una sola variable puede tomar múltiples valores específicos, utilizamos la estructura **CASO** (también conocida como *switch* en lenguajes como Java o C#).

## 2. Características Principales
A diferencia del condicional simple, la estructura múltiple tiene reglas claras:

*   **Solo funciona con la igualdad:** No evalúa si algo es "mayor que" o "menor que". Compara si el valor actual de la variable es exactamente igual a una de las opciones.
*   **Tipos de datos compatibles:** Solo se pueden comparar variables de tipos discretos (Enteros, Caracteres o Cadenas).
*   **Estructura Limpia:** Permite listar todas las opciones de forma vertical y organizada.

## 3. Sintaxis del Caso
La estructura comienza con la palabra **Caso**, seguida de la variable a evaluar, y termina siempre con un **Fin Caso**.

```pseudocode
Caso (nombre_variable)
    valor1: 
        // Bloque de instrucciones 1
    valor2: 
        // Bloque de instrucciones 2
    Sino: 
        // [Opcional] Se ejecuta si no coincidió con nada
Fin Caso
```

> [!TIP]
> **Dato Curioso:** En programación, cuando algo aparece entre corchetes `[ ]` en los manuales, significa que es opcional. El bloque **Sino** dentro de un caso no es obligatorio, pero es una excelente práctica para manejar errores o valores inesperados.

## 4. Ejercicio Práctico: Días de la Semana
**Reto:** Dado un número del 1 al 7, determine a qué día de la semana corresponde (donde 1 es Lunes y 7 es Domingo).

### Algoritmo en Pseudocódigo:
```pseudocode
Algoritmo DiasDeLaSemana
    Variables: 
        numDia: Entero
        nombreDia: Cadena
Inicio
    Escribir "Indique el número del día (1-7):"
    Leer numDia

    Caso numDia
        1: nombreDia <- "Lunes"
        2: nombreDia <- "Martes"
        3: nombreDia <- "Miércoles"
        4: nombreDia <- "Jueves"
        5: nombreDia <- "Viernes"
        6: nombreDia <- "Sábado"
        7: nombreDia <- "Domingo"
        Sino:
           nombreDia <- "No existe"
    Fin Caso

    Escribir "El día es: ", nombreDia
Fin
```

---

## 5. Análisis de la ejecución
*   **Entrada "3":** El programa busca la opción 3 en la lista, asigna "Miércoles" a la variable y salta directamente al final del caso para mostrar el resultado.
*   **Entrada "8":** Como el 8 no está en la lista de opciones (1 al 7), el flujo cae en el bloque **Sino**, asignando el mensaje "No existe".
